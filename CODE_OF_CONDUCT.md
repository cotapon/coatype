# Contributor Covenant 行動規範

## 私たちの誓い

私たちメンバー、コントリビューター、およびリーダーは、年齢、体型、目に見えるまたは目に見えない障がい、民族性、性別の特徴、性自認と性表現、経験の度合い、教育水準、社会経済的地位、国籍、容姿、人種、宗教、性的指向にかかわらず、コミュニティへの参加が誰にとってもハラスメントのない体験となるようにすることを誓います。

## 私たちの基準

コミュニティにとって良い環境に貢献する行動の例:

- 他者への思いやりと敬意を示す
- 異なる意見・視点・経験を尊重する
- 建設的なフィードバックを与え、快く受け入れる
- 過ちの責任を受け入れ、影響を受けた人々に謝罪し、その経験から学ぶ
- 個人だけでなくコミュニティ全体にとって最善のことに目を向ける

許容されない行動の例:

- 性的な言葉や表現、あらゆる種類の性的な誘いや関心の表明
- 煽り、侮辱的・軽蔑的なコメント、個人的・政治的な攻撃
- 公的・私的を問わないハラスメント
- 明示的な許可なく他者の個人情報（物理的・電子的な住所など）を公開すること
- 職業上不適切と合理的に考えられるその他の行為

## 施行責任

コミュニティのリーダーは、許容される行動の基準を明確にし施行する責任を負い、不適切・脅迫的・攻撃的または有害と判断した行動に対して、適切かつ公正な是正措置を講じます。

## 適用範囲

この行動規範は、すべてのコミュニティの場、および個人がコミュニティを公式に代表する場面に適用されます。

## 施行

虐待的・嫌がらせ的・その他許容されない行動は、リポジトリオーナー（[@cotapon](https://github.com/cotapon)）に報告できます。すべての苦情は速やかかつ公正に検討・調査されます。
